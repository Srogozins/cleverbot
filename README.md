cleverbot
=========

A python3 binding of cleverbot wich **support special characters**.

How to
------

### Library

Example of how to use the library:

```python
import cleverbot

myBot = cleverbot.Session()
answer = myBot.Ask("Hello there")
print(answer)
```

### Command Line

Just run

```bash
python3 cleverbot.py
```
